Four Weeks at Perficient

Self Reflection

Technical
Strength Gained

Built fluency moving across Python, PySpark, SQL, and Power BI on the same dataset — comfortable designing a medallion architecture (Bronze/Silver/Gold) and tracing a single record's lineage end-to-end across 4 stack layers.

Challenge Faced

Hit dialect conflicts running sqlfluff across PostgreSQL and Databricks SQL constructs in CI — had to learn how linting rules differ per dialect rather than treating SQL as one language.

Area to Strengthen

Want to go deeper on Spark performance tuning (partitioning, shuffle behavior) for large-scale jobs beyond the 9K-record scale I worked with here.

Project Management
Strength Gained

Learned to right-size process to project complexity — ran Arya Lifeline lean (2 epics, 10 issues, 3 days) and BFSI structured (4 epics, ~38 issues, 5 milestones, 1.5 weeks) instead of using one template for both.

Challenge Faced

Underestimated how much time milestone sequencing (cleaning → testing → documentation → SQL → tasks) needs upfront — had to re-order a few issues mid-sprint once dependencies became clear.

Area to Strengthen

Want to get better at estimating issue-level effort before sprint start, rather than adjusting scope reactively once work is underway.

Soft Skills
Strength Gained

Got comfortable presenting technical decisions (architecture choices, validation logic) in a way non-technical reviewers could follow without losing precision.

Challenge Faced

Sometimes held back in team discussions until fully sure of a point, which occasionally meant good input came in later than it should have.

Area to Strengthen

Actively working on speaking up earlier with partial confidence, and treating team reviews as a place to think out loud rather than only to present finished answers.

Shout-Out

Grateful to my mentors and peers at Perficient for patient code reviews, sharp feedback, and making a four-week sprint feel genuinely collaborative. The structured approach to project tracking, emphasis on test coverage and linting, and the culture of learning-through-doing left a mark. Looking forward to carrying these practices forward.
